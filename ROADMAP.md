# Roadmap

The roadmap is evidence-driven and maintained by the Queen Coordinator. It does not prescribe fixed releases in advance.

## Current stage

Create the first complete **Platform + Challenge Slice** from the clean baseline.

The first issue must let the Worker Swarm decide the exact concept and technical approach inside these constraints:

- one real mobile-first discovery entry point;
- one original complete 2D, 2.5D, or 3D challenge;
- polished graphics and purposeful animation;
- Arabic RTL, English, and Turkish;
- result, replay, strict share link, friend attempt, comparison, and share-again;
- accessibility and reduced motion;
- current tests, build, and deployable preview;
- no accounts, payments, public feed, public UGC, or generic engine.

## After each merge

The Queen records the largest constraint and selects one next delivery type:

- Platform Slice;
- Challenge Slice;
- Platform + Challenge Slice;
- Repair Slice.

No second active implementation issue is permitted.
