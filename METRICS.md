# Metrics

Metrics implementation is deferred until the first end-to-end viral loop exists.

Planned privacy-safe funnel events:

- challenge_viewed
- challenge_started
- challenge_completed
- result_viewed
- share_attempted
- share_completed
- shared_link_opened
- friend_completed
- comparison_viewed
- share_again_attempted

Do not add a third-party analytics destination without an explicit decision.
